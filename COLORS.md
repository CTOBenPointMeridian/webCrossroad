# Crossroads Healing Center - Color Palette

## Primary Colors

### Deep Blue-Green (Primary)
- **Hex**: #0E3744
- **RGB**: 14, 55, 68
- **Usage**: Main branding, headings, primary CTAs, headers
- **Purpose**: Trust, stability, calming - perfect for healthcare/wellness

### Warm Off-White (Background)
- **Hex**: #F4F3EB
- **RGB**: 244, 243, 235
- **Usage**: Main background, light sections, neutral space
- **Purpose**: Soft, welcoming, approachable aesthetic

## Secondary Colors

### Light Blue-Green (Accent)
- **Hex**: #1a5a6d
- **RGB**: 26, 90, 109
- **Usage**: Secondary buttons, highlights, hover states
- **Purpose**: Lighter variant for depth and hierarchy

### Muted Teal (Soft Accent)
- **Hex**: #8B9EA3
- **RGB**: 139, 158, 163
- **Usage**: Borders, secondary text, subtle accents
- **Purpose**: Sophisticated, not overwhelming

### Warm Tan (Accent)
- **Hex**: #D4C5B0
- **RGB**: 212, 197, 176
- **Usage**: Decorative elements, alternative accents
- **Purpose**: Adds warmth and sophistication

## Neutral Colors

### Warm Neutral Border
- **Hex**: #d9d3c8
- **RGB**: 217, 211, 200
- **Usage**: Dividers, borders, subtle separation
- **Purpose**: Warm alternative to cold grays

### Muted Text
- **Hex**: #6b7a80
- **RGB**: 107, 122, 128
- **Usage**: Secondary text, descriptions, metadata
- **Purpose**: Readable without harsh contrast

### Dark Blue-Green (Dark Mode Primary)
- **Hex**: #092534
- **RGB**: 9, 37, 52
- **Usage**: Dark mode backgrounds, deep shadows
- **Purpose**: Maintains brand while adapting to dark mode

## Usage Guidelines

### Backgrounds
- **Light**: Use #F4F3EB (Warm Off-White)
- **Dark**: Use #0E3744 (Deep Blue-Green) or #092534 (Darker variant)

### Text
- **Primary Text**: Use #0E3744 on light backgrounds
- **Secondary Text**: Use #6b7a80 (Muted Text)
- **Light Text**: Use #F4F3EB on dark backgrounds

### Accent Elements
- **Primary CTA**: #0E3744 (Deep Blue-Green)
- **Secondary CTA**: #1a5a6d (Light Blue-Green)
- **Borders**: #d9d3c8 (Warm Neutral)
- **Hover States**: #1a5a6d (Light Blue-Green)

### Gradients
- **Primary Gradient**: #0E3744 → #1a5a6d
- **Warm Gradient**: #F4F3EB → #D4C5B0
- **Trust Gradient**: #0E3744 → #092534

## CSS Variables (Available in Code)
```css
--primary: #0E3744
--primary-dark: #092534
--primary-light: #1a5a6d
--secondary: #F4F3EB
--accent: #8B9EA3
--accent-secondary: #D4C5B0
--border: #d9d3c8
--text-muted: #6b7a80
```

## Dark Mode
- Background: #0E3744 or #092534
- Text: #F4F3EB
- Borders: #1a5a6d
- Accents: #8B9EA3 or #D4C5B0

## Accessibility
All color combinations meet WCAG AA standard for contrast ratios:
- #0E3744 text on #F4F3EB background: 11.8:1 ✓
- #6b7a80 text on #F4F3EB background: 6.2:1 ✓
- #F4F3EB text on #0E3744 background: 11.8:1 ✓
